## Responsive classes

The border color and border side includes six tiers of predefined classes for building complex responsive colors and sides. Customize the color or sides of your component on extra small, small, medium, large, or extra large devices however you see fit.

-   `sm:u-borderX` - To use at the smallest Media Query breakpoint.
-   `md:u-borderX` - To use at the medium Media Query breakpoint.
-   `lg:u-borderX` - To use at the large Media Query breakpoint.
-   `xl:u-borderX` - To use at the extra large or desktop Media Query breakpoint.
-   `xxl:u-borderX` - To use at the monitor Media Query breakpoint.
