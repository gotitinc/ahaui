# Split Pane

## Introduction

Using the Split Pane component for creating and resizing multi-view layouts. It allows the layouts to be displayed as the viewport width or height increases.

This component is based on [react-split-pane][1]

## Examples

### Horizontal

<CodeView codeText={`
<div className="u-flex u-flexColumn u-paddingExtraSmall u-positionRelative" style={{height:320}}>
  <SplitPane split="horizontal" paneStyle={{
    minHeight: 'calc(30% - 8px)',
    maxHeight: 'calc(70% - 8px'
  }}>
    <div className="u-border u-paddingExtraSmall u-widthFull u-flex u-alignItemsCenter u-justifyContentCenter">Top</div>
    <div className="u-border u-paddingExtraSmall u-widthFull u-heightFull u-flex u-alignItemsCenter u-justifyContentCenter">Bottom</div>
  </SplitPane>
</div>`}
/>
