# Flexbox

Quickly manage the layout, alignment, and sizing of grid columns, navigation, components, and more with a full suite of responsive flexbox utilities. For more complex implementations, custom CSS may be necessary

## Base

Use `.u-flex` or `.u-flexInline` to enable flex behaviors

<CodeView exampleClassName="example-flexbox" codeText={`<div class="u-flex u-paddingExtraSmall b-highlight">I'm a flexbox container!</div>`}/>
<CodeView exampleClassName="example-flexbox" codeText={`<div class="u-flexInline u-paddingExtraSmall b-highlight">I'm a inline flexbox container!</div>`}/>

## Direction

Use `.u-flexRow` to set a horizontal direction (the browser default), or `.u-flexRowReverse` to start the horizontal direction from the opposite side

<CodeView exampleClassName="example-flexbox" codeText={directionRow}/>

Use `.u-flexColumn` to set a vertical direction, or `.u-flexColumnReverse` to start the vertical direction from the opposite side.

<CodeView exampleClassName="example-flexbox" codeText={directionColumn}/>

## Justify content

Use `.u-justifyContent{X}` on flexbox containers to change the alignment of flex items on the main axis (the x-axis to start, y-axis if flex-direction: column). Choose `X` from Start (browser default), End, Center, Between, or Around.

<CodeView exampleClassName="example-flexbox" codeText={justifyContent}/>

## Align items

Use `.u-alignItems{X}` on flexbox containers to change the alignment of flex items on the cross axis (the x-axis to start, y-axis if flex-direction: column). Choose `X` from Start, End, Center, Baseline, or Stretch(browser default).

<CodeView exampleClassName="example-flexbox" codeText={alignItems}/>

## Align self

Use `.u-alignSelf{X}` on flexbox items to individually change their alignment on the cross axis (the y-axis to start, x-axis if flex-direction: column). Choose `X` from the same options as align-items: Start, End, Center, Baseline, or Stretch (browser default)

<CodeView exampleClassName="example-flexbox" codeText={alignSelf}/>

## Wrap

Change how flex items wrap in a flex container. Choose from no wrapping at all (the browser default) with `.u-flexNoWrap`, wrapping with `.u-flexWrap`, or reverse wrapping with `.u-flexWrapReverse`.

<CodeView exampleClassName="example-flexbox" codeText={wrapFlexbox}/>
<CodeView exampleClassName="example-flexbox" codeText={wrapFlexbox2}/>

## Align content

Use `.u-alignContent{X}` on flexbox containers to align flex items together on the cross axis. Choose `X` from Start (browser default), End, Center, Between, Around, or Stretch. To demonstrate these utilities, we’ve enforced flex-wrap: wrap and increased the number of flex items.

<CodeView exampleClassName="example-flexbox" codeText={alignContent}/>

## Fill

Use the `.u-flexFill` class on a series of sibling elements to force them into widths equal to their content (or equal widths if their content does not surpass their border-boxes) while taking up all available horizontal space.

## Responsive classes

The flexbox includes six tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.

* `sm:u-X`  - To use at the smallest Media Query breakpoint.
* `md:u-X`  - To use at the medium Media Query breakpoint.
* `lg:u-X`  - To use at the large Media Query breakpoint.
* `xl:u-X`  - To use at the extra large or desktop Media Query breakpoint.
* `xxl:u-X`  - To use at the monitor Media Query breakpoint.
