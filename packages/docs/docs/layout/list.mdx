# List Style

## Introduction

Unordered lists (`<ul>`) - the list items are marked with bullets.
Ordered lists (`<ol>`) - the list items are marked with numbers or letters.

All list are customized to remove their left padding and put the bullets inside of the box by adding  `.u-paddingLeftNone`  and  `.u-listStylePositionInside`  utilities classes.
