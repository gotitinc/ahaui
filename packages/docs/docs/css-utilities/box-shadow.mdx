# Box Shadow

export const classExtractBoxShadow = [
  { className: "u-shadowSmall" },
  { className: "u-shadowMedium" },
  { className: "u-shadowLarge" },
  { className: "u-shadowExtraLarge" },
  { className: "u-shadowNone" },
  { className: "focus:u-shadowOutline", clickAble: true },
];

## Introduction

Add or remove shadows to elements with box-shadow CSS-Utilities.

## Hover

Convey meaning through opacity with a handful of shadow utility classes. Includes support for styling links with hover states, too.

Example: &nbsp;<code className="hover:u-shadowLarge">.hover:u-shadowLarge</code>
