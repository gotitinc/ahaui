# Opacity

import classNames from 'classnames';

## Introduction

Use the CSS-Utility opacity to control the opacity of an element.

## Hover

Convey meaning through opacity with a handful of opacity utility classes. Includes support for styling links with hover states, too.

Example: &nbsp;<code className="hover:u-opacityThreeQuarter">.hover:u-opacityThreeQuarter</code>
