# Position

## Examples

### Relative and Absolute

Absolute (`.u-positionAbsolute`) elements should be in side a relative (`.u-positionRelative`) element.

### Corners
* Top-Left: `.u-positionAbsolute .u-positionTop .u-positionLeft`
* Top-Right: `.u-positionAbsolute .u-positionTop .u-positionRight`
* Bottom-Left: `.u-positionAbsolute .u-positionBottom .u-positionLeft`
* Bottom-Right: `.u-positionAbsolute .u-positionBottom .u-positionRight`
* Center: `.u-positionAbsolute .u-positionCenter`
* Horizontal Center: `.u-positionAbsolute .u-positionHorizontalCenter`
* Vertical Center: `.u-positionAbsolute .u-positionVerticalCenter`
* Full: `.u-positionAbsolute .u-positionFull`
