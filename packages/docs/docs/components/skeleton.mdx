# Skeleton

## Introduction

Display a placeholder preview of the content before the data gets loaded to reduce load-time frustration. The Skeleton can be used to illustrate the layout of a component or block content.

By default, a single Skeleton item is basically a gray block of which the width and the height are set to 100% and 16px respectively.

## Examples

### Text

The default shape of the Skeleton and can be used to illustrate a text line. Two or more shapes can be used to represent a paragraph.

<CodeView codeText={"<Skeleton />"} />

### Circle

Often used to illustrate an Avatar.

<CodeView codeText={'<Skeleton variant="circle" width={40} height={40} />'} />

### Rectangle

To represent the media content such as image and video.

<CodeView codeText={"<Skeleton height={118} />"} />
