<ol className="u-paddingLeftLarge u-textGray u-marginTopTiny">
  <li>
    Label &nbsp;
    <i className="u-text200">
      - Short descriptive text which corresponds to the information required
      inside the field.
    </i>
  </li>
  <li>
    Placeholder&nbsp;
    <i className="u-text200">
      - The placeholder specifies a short hint that describes the expected value
      of an input field (i.e. a sample value or a short description of the
      expected format).
    </i>
  </li>
  <li>
    Value&nbsp;
    <i className="u-text200">
      - The value specifies the text information, data or value which is entered
      by users.
    </i>
  </li>
  <li>
    Feedback&nbsp;
    <i className="u-text200">
      - The feedback displays the messages from the system in specific cases
      i.e. invalid input, valid input, etc.
    </i>
  </li>
  <li>
    Input border&nbsp;
    <i className="u-text200">
      - The input border attribute specifies different input states (hovered,
      invalid, valid, etc.) by colors.
    </i>
  </li>
  <li>
    Input background&nbsp;
    <i className="u-text200">
      - The standard input background is white. It will only be changed to grey
      when the field is disabled.
    </i>
  </li>
</ol>

## Form Group

Input group wraps inputs and buttons for more complex components.

Place one add-on or button on either side of an input. You may also place one on both sides of an input. Remember to place `<label>` above the input group.

## Design Reference

Don’t use multiple lines for label & feedback

<div className="Grid u-marginBottomLarge">
  <div className="u-sizeFull md:u-size1of2">
    <Form.Group controlId="exampleForm.DSRe1">
      <Form.Label>Input</Form.Label>
      <Form.Input isValid type="text"></Form.Input>
      <Form.Feedback>Lorem Ipsum is simply dummy text</Form.Feedback>
    </Form.Group>
    <div className="u-borderBottom u-borderLarge u-borderPositive u-marginTopSmall u-paddingTopMedium" />
    <div className="u-textPositive">Do</div>
  </div>
  <div className="u-sizeFull md:u-size1of2">
    <Form.Group controlId="exampleForm.DSRe2">
      <Form.Label>Input</Form.Label>
      <Form.Input isInvalid type="text"></Form.Input>
      <Form.Feedback type="invalid">
        Lorem Ipsum is simply dummy text
      </Form.Feedback>
      <Form.Feedback type="invalid">
        Lorem Ipsum is simply dummy text
      </Form.Feedback>
    </Form.Group>
    <div className="u-borderBottom u-borderLarge u-borderNegative u-marginTopSmall" />
    <div className="u-textNegative">Don't</div>
  </div>
</div>

Avoid too long placeholder, label & feedback
