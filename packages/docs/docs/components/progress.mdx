# Progress

## Introduction

The component is used to help users to indicate the current state; and guide them through a set of steps in a specific process.

<img
  src={StbProgressStructure1}
  srcSet={`${StbProgressStructure2} 2x`}
  alt="Progress structure"
/>
<ol className="u-textGray u-marginTopTiny">
  <li>
      Indicator
    <i className="u-text200"> - An animated or fixed bar which is used to indicate the current state of the process.</i>
  </li>
  <li>
      Label
    <i className="u-text200"> - A text object, which is used to shortly describe the current state (i.e. percentage).</i>
  </li>
  <li>
      Progress bar
    <i className="u-text200">
        - A fixed-width bar, with outlined or secondary colors, that indicates the whole process.
    </i>
  </li>

</ol>
