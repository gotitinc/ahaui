# Breadcrumb

## Introduction

Indicate the current page’s location within a navigational hierarchy.

<img
  src={StbBreadcrumbStructure1}
  srcSet={`${StbBreadcrumbStructure2} 2x`}
  alt="Breadcrumb structure"
/>
<ol className="u-textGray u-marginTopTiny">
  <li>
    Breadcrumb items
    <i className="u-text200">
      {" "}
      - Indicates pages in hierarchical order. Items are basically button links,
      except for the current page item.
    </i>
  </li>
  <li>
    Separator
    <i className="u-text200"> - Icon which is used to separate items.</i>
  </li>
</ol>
