# Multi Steps

## Introduction

The component is used when a progress needs to be divided into multiple logical and numbered steps.

<img
  src={StbMultiStepsStructure1}
  srcSet={`${StbMultiStepsStructure2} 2x`}
  className="u-maxWidthFull"
  alt="MultiSteps"
/>
<ol className="u-textGray u-marginTopTiny">
  <li>
      Step
    <i className="u-text200"> - Display step number or current state of the step.</i>
  </li>
  <li>
      Label
    <span className="u-marginLeftTiny u-text200">(optional)</span>
    &nbsp;
    <i className="u-text200"> - Display short & descriptive name/title of the step.</i>
  </li>
  <li>
      Progress bar
    <i className="u-text200"> - Indicate the current progress of the process.</i>
  </li>
  <li>
      Substrate
    <i className="u-text200"> -  Indicate the full achievable process.</i>
  </li>
</ol>
