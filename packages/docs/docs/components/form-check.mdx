# Checkbox and Radio Button

## Introduction

### Checkbox

A checkbox is a square box that is checked when activated. Checkboxes allow users to choose multiple options over a series of options.

### Radio Button

A radio button is a small circle button that is checked when activated. Radio buttons allow users to choose only one option over a series of options.
