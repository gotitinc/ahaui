# Composer


## Introduction

Displaying at the bottom of the chat window, the Composer component helps users to enter and edit the text messages, and also, attach files.

  src={StbComposerStructure1}

/>
<ol className="u-textGray u-marginTopTiny">

    </i>
  </li>
  <li>
    File attachment
    <i className="u-text200">
      - A button that lets users attach files.
    </i>
  </li>
  <li>
    Top divider

    </i>

</ol>
